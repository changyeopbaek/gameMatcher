# 배드민턴 게임 매칭 웹사이트

배드민턴 복식 게임을 공정하고 균등하게 매칭해주는 웹 애플리케이션입니다.

## 기능

### 필수 기능
- 참여자 모두가 거의 동일한 게임 수를 가지도록 자동 매칭
- 남자/여자 참가자 구분 입력
- 코트 수, 운동 시간, 시간당 게임 수 설정

### 선택 기능
- **혼복 허용 여부**: 남복 & 여복만으로 게임 구성 또는 혼복 포함
- **파트너 고정**: 향후 구현 예정

## 사용 방법

1. 프로젝트 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:5173` 접속

## 매칭 알고리즘

1. **기본 계산**
   - 총 게임 수 = 코트 수 × 운동 시간 × 시간당 게임 수
   - 총 슬롯 수 = 총 게임 수 × 4 (복식이므로)
   - 1인당 목표 게임 수 = floor(총 슬롯 수 / 참가자 수)

2. **팀 생성**
   - 혼복 미허용: 남복 팀과 여복 팀만 생성
   - 혼복 허용: 남복, 여복, 혼복 팀 모두 생성

3. **매칭 스케줄링**
   - 라운드별로 코트에 게임 배정
   - 각 참가자의 출전 횟수를 균등하게 유지
   - 같은 매치 반복 최소화
